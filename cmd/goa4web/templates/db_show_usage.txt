Usage:
  {{.Prog}} db show <file>

The db show command displays the contents of the embedded SQL files that are
compiled into the application. This is useful for inspecting the raw SQL that is
used to create the database schema and to seed the database with initial data.

Arguments:
  seed.sql           - Show the contents of the seed.sql file.
  schema.mysql.sql   - Show the contents of the schema.mysql.sql file.
  schema-version     - Show the current schema version from the database.

Examples:
  # Show the contents of the seed.sql file
  {{.Prog}} db show seed.sql

  # Show the contents of the schema.mysql.sql file
  {{.Prog}} db show schema.mysql.sql

  # Show the current schema version
  {{.Prog}} db show schema-version
