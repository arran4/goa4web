Usage: goa4web role <subcommand> [flags]

Subcommands:
  load   Loads a role from a seed script.
  reset  Resets a role to the state defined in its seed script.
  apply  Applies the grants from a source role to a destination role.
  remove Removes the grants from a destination role that match a source role.
  users  Lists the users in a role.
  list   Lists available embedded role seed scripts.

Examples:
  # Load the "content-writer" role from its seed script
  {{.Prog}} role load -role content-writer

  # Load from a custom SQL file instead of the embedded script
  {{.Prog}} role load -role content-writer -file ./my-content-writer.sql

  # Reset the "moderator" role to its default state
  {{.Prog}} role reset -role moderator

  # Apply the permissions of the "private-forum-user" role to the "user" role
  {{.Prog}} role apply -src private-forum-user -dest user

  # Remove the permissions of the "private-forum-user" role from the "user" role
  {{.Prog}} role remove -src private-forum-user -dest user

  # Remove all grants from the "user" role
  {{.Prog}} role remove -dest user

  # List embedded roles available in this build
  {{.Prog}} role list
