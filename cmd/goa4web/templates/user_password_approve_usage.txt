  {{.Prog}} user password approve --code <verification_code>

Description:
  Approve a pending password change by username, ID, or verification code.
  If a user has multiple pending password changes, the --code flag must be used to specify which one to approve.

Examples:
  {{.Prog}} user password approve alice
  {{.Prog}} user password approve -username alice
  {{.Prog}} user password approve -id 3
  {{.Prog}} user password approve --code 1234567890abcdef
