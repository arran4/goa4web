Usage:
  {{.Prog}} user password list [flags]

The user password list command lists password reset requests, ordered from most
recent to oldest. Results can be filtered and paginated.

Examples:
  # List pending reset requests
  {{.Prog}} user password list

  # List verified resets for a specific user
  {{.Prog}} user password list -status verified -user alice

  # List resets created since a specific time
  {{.Prog}} user password list -since 2024-01-01T00:00:00Z

  # Output results in JSON
  {{.Prog}} user password list -json
