Usage:
  {{.Prog}} user password [flags]

The user password command is used to manage a user's password. You can use this
command to set a new password for a user.

You must specify the user by their username and provide the new password.

Examples:
  # Set a new password for the user "bob"
  {{.Prog}} user password -username bob -password "newsecret"
