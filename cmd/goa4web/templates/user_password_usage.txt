Usage:
  {{.Prog}} password <command> [<args>]

The password command is used to manage user password reset requests. You can
list, clear, and reset password requests to help users regain access to their
accounts.

Commands:
  list           List all pending password reset requests.
  clear-expired  Remove all expired password reset requests from the system.
  clear-user     Remove all password reset requests for a specific user.
  reset          Manually reset a user's password.

Examples:
  # List all pending password reset requests
  {{.Prog}} password list

  # Clear all expired password reset requests
  {{.Prog}} password clear-expired

  # Clear all password reset requests for the user "bob"
  {{.Prog}} password clear-user -username bob

  # Reset the password for the user "bob"
  {{.Prog}} password reset -username bob -password "new_secret"
