Usage:
  {{.Prog}} user password <command> [<args>]

The user password command manages password reset requests and administrative
password changes.

Commands:
  list           List password reset requests.
  approve        Approve a pending password reset request.
  clear-expired  Remove expired password reset requests.
  clear-user     Clear reset requests for a specific user.
  force-change   Force a password change for a user.
  generate-reset Generate a password reset link and temporary password.
  send-reset-email Send a reset email to a user.

Examples:
  # List pending password reset requests
  {{.Prog}} user password list

  # Approve a password reset by ID
  {{.Prog}} user password approve -id 42

  # Remove expired reset requests
  {{.Prog}} user password clear-expired -hours 24
